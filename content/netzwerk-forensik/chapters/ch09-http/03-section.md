<h2 class="section-title"><span class="number">9.3</span> HTTP-Statuscodes</h2><p>Der Statuscode ist die erste Zeile der Antwort &ndash; er sagt dir sofort, ob alles OK ist:</p><div class="table-container"><table><thead><tr><th>Code</th><th>Bedeutung</th><th>Einfach gesagt</th></tr></thead><tbody><tr><td><strong>200</strong></td><td>OK</td><td>Alles gut, hier sind deine Daten</td></tr><tr><td><strong>301</strong></td><td>Moved Permanently</td><td>Dauerhaft umgezogen (neue URL)</td></tr><tr><td><strong>302</strong></td><td>Found</td><td>Vor&uuml;bergehend umgeleitet</td></tr><tr><td><strong>400</strong></td><td>Bad Request</td><td>Der Request ist fehlerhaft</td></tr><tr><td><strong>401</strong></td><td>Unauthorized</td><td>Bitte erst anmelden</td></tr><tr><td><strong>403</strong></td><td>Forbidden</td><td>Du darfst das nicht sehen</td></tr><tr><td><strong>404</strong></td><td>Not Found</td><td>Seite existiert nicht</td></tr><tr><td><strong>500</strong></td><td>Internal Server Error</td><td>Server hat ein Problem</td></tr><tr><td><strong>503</strong></td><td>Service Unavailable</td><td>Server &uuml;berlastet / Wartung</td></tr></tbody></table></div><p><strong>Faustregel:</strong> <span class="inline-code">2xx</span> = OK, <span class="inline-code">3xx</span> = Umleitung, <span class="inline-code">4xx</span> = Dein Fehler, <span class="inline-code">5xx</span> = Server-Fehler</p>
