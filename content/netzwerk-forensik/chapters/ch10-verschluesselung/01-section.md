<h2 class="section-title"><span class="number">10.1</span> Warum Verschl&uuml;sselung?</h2><p>Stell dir vor, du schickst eine Postkarte: Jeder Postmitarbeiter kann sie lesen. Eine Netzwerkverbindung ohne Verschl&uuml;sselung ist genau das &ndash; jeder im selben WLAN oder zwischen dir und dem Server kann mitlesen.</p><p><strong>Verschl&uuml;sselung</strong> verwandelt lesbaren Text (Klartext) in unlesbaren Code (Chiffretext). Nur wer den richtigen Schl&uuml;ssel hat, kann ihn wieder entschl&uuml;sseln.</p><blockquote><p><strong>Merke:</strong> Ohne Verschl&uuml;sselung sind Passw&ouml;rter, Cookies, Formulardaten und alles andere im Klartext &uuml;ber das Netzwerk sichtbar. Das ist der Grund warum <strong>HTTPS</strong> statt HTTP der Standard ist.</p></blockquote>
