<h2 class="section-title"><span class="number">13.2</span> Die Routing-Tabelle</h2><p>Die Routing-Tabelle sagt dem System, wohin Pakete geschickt werden sollen. Jede Zeile ist eine Regel.</p><div class="code-block"><div class="code-header"><span class="lang">BASH</span><button class="copy-btn">Kopieren</button></div><pre><code># Routing-Tabelle anzeigen
ip route show</code></pre></div><p>Typische Ausgabe:</p><div class="code-block output-block"><div class="code-header"><span class="lang">ERWARTETE AUSGABE</span></div><pre><code>default via 192.168.1.1 dev eth0       # Default-Route (Alles Unbekannte)
192.168.1.0/24 dev eth0 proto kernel   # Lokales Netz (direkt erreichbar)</code></pre></div><div class="table-container"><table><thead><tr><th>Eintrag</th><th>Bedeutung</th><th>Einfach gesagt</th></tr></thead><tbody><tr><td><span class="inline-code">default</span></td><td>Default-Route</td><td>Alles was nicht anders passt &rarr; zum Gateway</td></tr><tr><td><span class="inline-code">via 192.168.1.1</span></td><td>N&auml;chster Hop</td><td>&Uuml;ber diesen Router gehen die Pakete</td></tr><tr><td><span class="inline-code">dev eth0</span></td><td>Interface</td><td>&Uuml;ber diese Netzwerkkarte</td></tr><tr><td><span class="inline-code">proto kernel</span></td><td>Quelle</td><td>Vom System automatisch eingetragen</td></tr></tbody></table></div><blockquote><p><strong>Merke:</strong> Wenn ein Paket nicht ins lokale Netz passt, geht es an die Default-Route (= dein Gateway). Das ist der Ausgang ins Internet.</p></blockquote>
