<h2 class="section-title"><span class="number">13.2</span> Software-Write-Blocker</h2><p>Kernel-Level Protection, die Writes blockt.</p><div class="code-block"><div class="code-header"><span class="lang">BASH</span><button class="copy-btn">Kopieren</button></div><pre><code># Linux Kernel Write-Blocker (SysRq)
echo 1 > /proc/sys/kernel/sysrq
echo w > /proc/sysrq-trigger

# Pr&uuml;fen ob RO gesetzt ist
lsblk -o NAME,RO

# Reset (nur f&uuml;r Entwicklungs-Systeme!)
echo 0 > /proc/sys/kernel/sysrq</code></pre></div><div class="table-container"><table><thead><tr><th>Option</th><th>Beschreibung</th></tr></thead><tbody><tr><td>1</td><td>Alle SysRq-Funktionen aktivieren</td></tr><tr><td>w</td><td>Alle Schreibzugriffe blocken (Write-Blocker)</td></tr><tr><td>0</td><td>SysRq deaktivieren</td></tr></tbody></table></div><div class="callout callout-danger"><div class="callout-header">&#9888;
