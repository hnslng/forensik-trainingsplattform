<h2 class="section-title"><span class="number">8.1</span> Was ist DHCP?</h2><p><strong>DHCP</strong> (Dynamic Host Configuration Protocol) = Automatische Vergabe von IP-Adressen. Stell dir vor, jeder Gast in einem Hotel bekommt automatisch ein Zimmer zugewiesen &ndash; genau das macht DHCP f&uuml;r Ger&auml;te im Netz.</p><p><strong>Ohne DHCP</strong> m&uuml;sstest du jedem Ger&auml;t manuell eine IP-Adresse, Subnetzmaske, Gateway und DNS-Server eintragen. Bei 50 Ger&auml;ten im B&uuml;ro ein Albtraum.</p><p><strong>Was DHCP verteilt:</strong></p><ul><li>IP-Adresse + Subnetzmaske</li><li>Default-Gateway (Router)</li><li>DNS-Server</li><li>Optional: NTP-Server, Domain-Name</li></ul>
