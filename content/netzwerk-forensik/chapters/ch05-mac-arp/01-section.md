<h2 class="section-title"><span class="number">5.1</span> Was ist eine MAC-Adresse?</h2><p><strong>MAC</strong> (Media Access Control) = Die Hardware-Adresse jedes Netzwerkger&auml;ts. Sie ist fest in der Netzwerkkarte gespeichert und weltweit eindeutig.</p><p>Stell dir vor, jedes Ger&auml;t im Netz hat eine feste Hausnummer &ndash; das ist die MAC-Adresse. Im Gegensatz zur IP-Adresse &auml;ndert sich die MAC nicht (au&szlig;er man manipuliert sie bewusst).</p><p><strong>Aufbau:</strong></p><ul><li>48 Bit (6 Byte), dargestellt als Hexadezimal: <span class="inline-code">AA:BB:CC:DD:EE:FF</span></li><li><strong>Erste 3 Byte</strong> (AA:BB:CC): Hersteller-Kennung (<strong>OUI</strong> = Organizationally Unique Identifier)</li><li><strong>Letzte 3 Byte</strong> (DD:EE:FF): Eindeutige Ger&auml;tekennung</li></ul><p><strong>Beispiel:</strong> <span class="inline-code">08:00:27:A1:B2:C3</span> &rarr; Hersteller <span class="inline-code">08:00:27</span> = Oracle (VirtualBox)</p><div class="code-block"><div class="code-header"><span class="lang">BASH</span><button class="copy-btn">Kopieren</button></div><pre><code># Eigene MAC-Adresse anzeigen
ip link show

# Nur MAC eines bestimmten Interfaces
ip link show eth0 | grep ether</code></pre></div><blockquote><p><strong>Merke:</strong> IP-Adresse = logische Adresse (kann sich &auml;ndern), MAC-Adresse = physische Adresse (fest).</p></blockquote>
